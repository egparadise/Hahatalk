# Media segmentation runtime

The files in this directory support on-device camera background processing.

- `vision_wasm_*`: MediaPipe Tasks Vision 0.10.14, Apache-2.0.
- `selfie_segmenter.tflite`: MediaPipe Selfie Segmenter model distributed by Google.
- Runtime integration: `@livekit/track-processors` 0.5.7, Apache-2.0.

Project sources:

- https://github.com/google-ai-edge/mediapipe
- https://github.com/livekit/track-processors-js
